# CKipTagger

#介紹

CkipTagger為中研院資訊所、語言所合作的中文計算語言研究小組所共同開發之中文處理套件。
系統以python寫成，效能優異，且呼叫方式簡潔、易於整合。而此工具之功能包含:斷詞(WOS)、詞性標註(POS)、實體辨識(NER)。
